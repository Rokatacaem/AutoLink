# Credenciales de Prueba AutoLink

Para probar la aplicación desde tu móvil, asegúrate de que ambos dispositivos estén en la misma red Wi-Fi.

## 1. Configuración

* **IP del Servidor:** `192.168.1.27` (Actualizada)
* **Puerto:** `8000`

## 2. Usuarios para Pruebas

Puedes registrar nuevos usuarios libremente en la App, pero aquí tienes el formato recomendado:

### A. Rol: Cliente (Dueño de Auto)

* **Email:** `cliente1@test.com`
* **Password:** `123456`
* **Acceso:** Verás "Mi Garaje" y podrás usar "AI Diagnosis".

### B. Rol: Mecánico (Partner)

* **Email:** `taller@mechanic.com`  (IMPORTANTE: debe contener "mechanic")
* **Password:** `123456`
* **Acceso:** Verás el "Inbox" de solicitudes de servicio.

## 3. Pasos para Probar Flujo Completo

1. Logueate como **Cliente**.
2. Ve a Diagnostics y envía un reporte (ej: "Frenos suenan mal").
3. Cierra sesión (o usa otro celular) y entra como **Mecánico**.
4. Deberías ver la solicitud del cliente en tu Inbox.
